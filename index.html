按钮

非常简单的CSS按钮样式

默认按钮

给input或者button添加class .btn可以实现按钮样式。

  1. <button class="btn blue">button</button>
  2. <input type="button" class="btn blue">

禁止按钮

给input或者button添加属性 disabled可以实现禁止按钮。

  1. <button class="btn" disabled>button</button>
  2. <input type="button" class="btn" disabled>

彩色按钮

给input或者button添加 class

.red.orange.yellow.green.blue.black.primary可以实现彩色按钮样式。

  1. <button class="btn red">button</button>
  2. <button class="btn orange">button</button>
  3. <button class="btn yellow">button</button>
  4. <button class="btn green">button</button>
  5. <button class="btn blue">button</button>
  6. <button class="btn black">button</button>
  7. <button class="btn primary">button</button>

按钮大小

给input或者button添加class.big.small可以实现不同大小按钮。

  1. <button class="btn blue big">button</button>
  2. <button class="btn blue">button</button>
  3. <button class="btn blue small">button</button>

表单

非常简单的CSS表单样式

默认输入框

给input添加class .input可以实现默认输入框样式。

给textarea添加class .textarea可以实现默认输入框样式。



  1. <input type="text" class="input"/>
  2. <textarea class="textarea"></textarea>

禁止输入框

给input添加属性 disabled可以禁止输入框。

  1. <input type="text" class="input" disabled/>

输入框组

给<form>添加class.input-group就可以创建输入框组。

  1. <form class="input-group">
  2.   <input type="text" class="input"/>
  3.   <button class="btn primary">Button</button>
  4. </form>

单行表单

给<form>添加class.form-inline就可以创建单行表单。

  1. <form class="form-inline">
  2.   <label>Username:</label>
  3.   <input type="text" class="input"/>
  4.   <button class="btn blue">Button</button>
  5. </form>

栅格化

十二分之一的网格布局

Grad

给<div>的子节点添加class .u-xs-*.u-sm-*.u-md-*.u-lg-*可以定义栅格。

u-xs-*代表手机下显示样式。u-sm-*代表平板下显示样式。u-md-*代表电脑屏幕下显示样式。u-lg-*代表大屏幕显示器下显示样式。

u-*-*共有数字1------12可选，数字相加超过12会另起到下一行。


自适应显隐

给<div>的子节点添加class .show-xs.show-sm.show-md.show-lg.hide-xs.hide-sm.hide-md.hide-lg可以控制不同分辨率下的显示和隐藏。

.hide-xs

.hide-sm

.hide-md

.hide-lg

.show-xs

.show-xs

.show-sm

.show-md

.show-lg


其他属性

给<div>的添加class .container.container-fluid可以界面模板的居中和边距。

*在栅格内部避免使用.container，而改用.container-fluid避免页面宽度溢出。

给<div>的添加class .clearfix可以清除浮动。
